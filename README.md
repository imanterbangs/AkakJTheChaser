# Akak J The Chaser

# Pengenalan

Kau sebagai mak rempit, dilihat oleh Akak J semasa kau angkat tayar depan tu
mcm batang kau naik time sejuk (sorry didn't meant to), setelah dikejar oleh Akak J,
kau pun lari sambil mengumpul barangan yang kau suka sangat tu lagi - lagi exzos yang
memecahkan gegendang telinga orang yang tak bersalah waktu malam.
Gunakan full screen untuk better gaming (ecehh) dengan cara tekan F11.

# Cara bermain
Project ini adalah open source, terpulang kepada anda untuk main dengan mana diantara 2 pilihan tersebut. Pertama,
download file ini dan bole main secara offline dengan cara drag file index.html ini ke browser. Kedua, layari laman sesawang
<a href="https://imanterbangs.github.io/" target="_blank">imanterbangs.github.io/</a>
